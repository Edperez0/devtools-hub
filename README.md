# DevTools Hub

Free collection of 16 essential developer tools. All tools run 100% client-side in your browser - your data never leaves your device.

## 🚀 Live Demo

[View Live Site](https://devtoolsub.com)

## 🛠️ Tools Included

1. **JSON Formatter** - Format, validate, and beautify JSON
2. **Base64 Encoder/Decoder** - Encode and decode Base64 strings
3. **URL Encoder/Decoder** - Encode and decode URLs
4. **Color Picker & Converter** - Convert between HEX, RGB, HSL
5. **UUID Generator** - Generate v4 UUIDs
6. **Hash Generator** - MD5, SHA-1, SHA-256, SHA-512
7. **Timestamp Converter** - Unix timestamp converter
8. **Regex Tester** - Test regular expressions
9. **Lorem Ipsum Generator** - Generate placeholder text
10. **QR Code Generator** - Create QR codes
11. **Password Generator** - Generate secure passwords
12. **Diff Checker** - Compare text differences
13. **Code Minifier** - Minify JS, CSS, HTML
14. **Code Beautifier** - Format and beautify code
15. **Image to Base64** - Convert images to Base64
16. **JWT Decoder** - Decode JSON Web Tokens

- ⚡ Lightning fast - all processing happens locally
